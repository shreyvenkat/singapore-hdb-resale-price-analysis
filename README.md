**HDB Resale Market Analysis**
Problem Statement

This project explores buyer demand patterns in the HDB resale market to guide a property company's brokerage strategy, examined across three angles:
**Sub Problems:**
1. High-Growth Towns :Which towns show rising resale prices and transaction volumes, pointing agents and clients toward high-opportunity locations?
2. Flat Age vs. Resale Price :How does flat age relate to resale price, and what does this mean for valuation strategy?
3. Impact of Cooling Measures :Did the government's Dec 2021 cooling measures make resale flats more affordable?

**Project Objectives**
Analyze HDB resale price trends over time
Compare resale prices across Singapore towns
Examine price differences between flat types
Study the relationship between floor area, remaining lease and resale price
Identify towns and flat types with the highest transaction volumes

## Dataset ##

The project uses publicly available Singapore HDB resale transaction data from the Singapore Government’s open-data portal.

**Key Analysis**

The analysis covers:

Data cleaning and missing-value handling
Exploratory data analysis
Resale price trends by year
Price comparison by town and flat type
Transaction volume analysis
Correlation between property features and resale prices
Identification of patterns, trends and outliers
